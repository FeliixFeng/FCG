# Family Care Guardian (FCG)

家庭健康管理系统 - 毕业设计项目

## 项目介绍
基于Spring Boot与Vue的家庭健康管家系统，针对家庭慢病护理与居家养老场景中的数据分散、适老化交互困难等问题，构建集用药管理、体征监护与亲情关怀于一体的家庭健康管理平台。

## 项目结构

```
fcg/
├── fcg-server/     # 后端服务 (Spring Boot + MyBatis-Plus)
└── fcg-client/     # 前端应用 (Vue 3 + Vite)
```

## 技术栈

### 后端
- Java 17
- Spring Boot 3.x
- MyBatis-Plus
- Maven
- MySQL 8.0

### 前端
- Vue 3
- Vite
- npm

## 开发环境

### 后端启动
```bash
cd fcg-server
# 使用IDEA打开，运行 FcgApplication.java
```

### 前端启动
```bash
cd fcg-client
npm install
npm run dev
```
